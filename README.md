# 💰 Richie is Rich

**The Official SaaS Landing Page for the 0.001%**

You know what's better than having a lot of money? Building a ridiculously over-engineered, high-fidelity dark-themed SaaS landing page just to flex that I can. 

Welcome to **Richie is Rich**, the financial platform built for the modern megalomaniac. 

## 🧐 Why did I build this?
Honestly? Because I was bored and my private jet was delayed by 15 minutes. 

I wanted to prove that I could drop a fully responsive, pixel-perfect, aesthetically impeccable Next.js application before my caviar got warm. And I did. The peasants can keep their generic templates; we build our own bespoke digital mansions here.

## 🏗️ The Flex Stack
I didn't just use standard tools; I used the *expensive* ones.
- **Next.js (App Router)**: Because raw React is for people who can't afford server-side rendering.
- **Tailwind CSS v4**: I don't write manual CSS like a serf. I use utility classes powered by the new Lightning CSS engine.
- **OKLCH Colors**: Hex codes? Please, that's so 2023. We only use perceptually uniform color spaces because our retinas demand absolute perfection. The primary accent is a vibrant `oklch(0.65 0.22 40)`.
- **Lucide Icons**: Because custom SVGs are too much effort when I have yachts to buy.
- **Bricolage Grotesque & Manrope**: System fonts are a tragedy. We use geometric display faces that scream "I have a hedge fund."

## ✨ Unreasonable Features I Added
- A **Dashboard Mockup** built entirely with CSS Grid and divs. No charting libraries. Why? Because importing a heavy library is a sign of weakness.
- A **"Money Printer"** code block that is literally just a `while (true)` loop incrementing net worth.
- **"Delight" Animations**: Hover states, pulses, floating effects, and an infinite marquee. The UI is literally breathing money.

## 🚀 How to Run (If your laptop can handle it)

1. **Clone this repository** (carefully, it's expensive).
2. **Install the dependencies**:
   ```bash
   npm install
   ```
3. **Start the money printer**:
   ```bash
   npm run dev
   ```
4. Open [http://localhost:3000](http://localhost:3000) and weep at its beauty.
